### 步骤3：安全边际与仓位

```
安全边际 JJ = GG − II = [值] pct

周期性修正（仅强周期，引用 Agent A 维度三）：
  底部 → II − 1pct
  顶部 → II + 2pct
  非周期或中段 → 不调整

修正后安全边际 KK = [值] pct
```

**仓位矩阵**：

| 条件 | 仓位建议 |
|------|---------|
| KK ≥ 1.5pct + 可信度高 + 陷阱低 | 标准仓位 |
| KK ≥ 1.5pct + 可信度高 + 陷阱中 | 70% |
| KK ≥ 1.5pct + 可信度中 + 陷阱低 | 70% |
| KK 0.5-1.5pct + 可信度高 + 陷阱低 | 50% |
| KK 0.5-1.5pct + 可信度中 + 陷阱低 | 观察（30%） |
| KK 0-0.5pct | 观察或等待 |
| KK < 0 | 不建仓 |
| 可信度低 | 观察或不建仓 |
| 陷阱高 + GG < II×1.5 | 排除 |

### 步骤4：股价位置与买入触发价

> §17.6 预计算加速：若存在，直接引用分位数据。

**4-1：目标买入价**
```
目标买入价 = 当前市值 × (GG / II) / 总股本
GG ≥ II → 目标买入价 ≥ 当前股价
GG < II → 目标买入价 < 当前股价
```

**4-2：历史价格数据**
```
数据源：§11/附录A（10年周线或全部上市以来）
NN = 数据点数
OO = 10年最低价（日期）
PP = 10年最高价（日期）
```

**4-3：分位与触发评估**
```
当前股价分位 = 低于当前价的数据点 / NN × 100%
目标买入价分位 = 低于目标价的数据点 / NN × 100%

触发概率：
  ≤10% → 极低  | 10-25% → 低  | 25-50% → 中等  | 50-75% → 较高  | >75% → 已达标
```

### 步骤5：绝对估值与基准价

> §17.8 预计算加速：若存在，直接引用。

估值指标：
- EV/EBITDA（< 8x → 偏低信号）
- 扣除现金PE（< 10x → 盈利能力强）
- FCF收益率（> 8% → 丰厚）
- 净负债/EBITDA（< 2x → 风险可控）

- \> 80% → 较高溢价
