## 维度一：商业模式与资本特征

**数据源**：§1, §3-5, §4P, §8

**分析内容**：

(1) **商业模式清晰度**
- 能否用一句话说清收入来源、成本结构、利润产生方式？
- 商业模式是否经过至少1个完整经济周期验证？
- 评价：[清晰且已验证 / 清晰但未充分验证 / 模糊]

(2) **资本消耗强度**
- 每年维持当前盈利规模的必要资本再投入金额
- 资本投入时间结构：一次投入长期受益 vs 每周期重新投入
- 以"维持每单位盈利所需消耗的持续性资本"为依据
- 评价：[capital-light / capital-hungry]

(3) **收款模式**
- 一笔典型交易的完整现金流时间线
- 垫资方向：企业垫资给客户 vs 占用供应商/客户资金
- 评价：[先款后货 / 订阅预收 / 先货后款 / 垫资回收]
- 对现金状况的影响：[正贡献 / 中性 / 负担]

**输出**：
```
商业模式清晰度：[评价]
资本消耗强度：[capital-light / capital-hungry]，依据（≤3点）
收款模式：[类型]，对现金影响 [正贡献/中性/负担]
```

## 维度二：竞争优势与护城河

**数据源**：§1, §8, §3-5

**分析内容**：

(1) **市场结构**：[垄断 / 寡头 / 垄断竞争 / 充分竞争]

(2) **护城河双层分析**

**第一层：非技术性护城河（商业壁垒）**
逐一判断（存在/不存在，强/中/弱）：
- 规模效应 | 网络效应 | 转换成本 | 无形资产（品牌/专利/牌照） | 成本优势

**第二层：技术性护城河（数据与算法壁垒）**
逐一判断（存在/不存在，强/中/弱）：
- 数据资产壁垒：专有数据集 + Data Flywheel？
- 核心算法/模型壁垒：复制调优周期 ≥ 2年？
- 履约/供应链系统壁垒：极端场景稳定性？
- AI/前沿技术投入：自有闭环数据训练的 AI？

> 并非所有企业都具备技术性护城河（传统制造业/消费品第二层可能为空），标注"不适用"即可。

**两层交互**：
- 技术层是否强化商业层？商业层是否反哺技术层？
- 若形成闭环飞轮 → 标注"复合护城河"

(3) **定价权**：主动提价记录，提价后客户流失率
(4) **产业链位置**：面对上下游的议价能力
(5) **护城河侵蚀风险**：过去5年是否有变窄迹象

(6) **人力资本依赖**
- 核心竞争力依赖关键人才的比例
- 竞争优势是否已沉淀为系统性能力
- 评价：[系统型 / 人才型]

**输出**：
```
市场结构：[类型]
护城河类型：[非技术] xxx + [技术] xxx（若不适用标注"不适用"）
复合护城河飞轮：[是/否]
定价权：[强/中/弱]
人力资本依赖：[系统型 / 人才型]
综合评价：[强 / 中 / 弱]
```

## 维度三：外部环境

**数据源**：§3, §8

**分析内容**：

(1) **周期性**
- 过去1-2个完整经济周期中收入和盈利的波动幅度（量化为百分比）
- 外部变量依赖度（大宗商品/利率/汇率）及传导机制
- 若为强周期：当前处于周期的 [底部 / 中段 / 顶部]
- 评价：[强周期 / 弱周期 / 非周期]

(2) **监管与政策风险**
- 商业模式对政策许可/监管保护的依赖度
- 行业监管历史与当前趋势（收紧/放开）
- 国资背景影响（若适用）
- 评价：[低风险 / 中风险 / 高风险]

**输出**：
```
周期性：[强周期 / 弱周期 / 非周期]
当前周期位置：[底部/中段/顶部/不适用]
监管风险：[低/中/高]
```

## 维度四：管理层与治理

**数据源**：§7, §12, §4P, data_pack_report P4

**分析内容**：

(1) **治理风险标志**
- 审计意见：最近一期是否异常（保留/否定/无法表示/持续经营重大不确定性）→ [正常 / ⚠️ 异常]
- 审计师变更：过去5年更换 ≥ 2次，尤其大所→小所 → [正常 / ⚠️ 频繁更换]
- 造假/违规前科：企业或控股股东有处罚/立案/谴责 → [无 / ⚠️ 有前科]
- 控股股东信号：质押率>70% / 诉讼 / 频繁减持 / 刑事调查 → [正常 / ⚠️ 负面信号]

(2) **管理层能力评估**
- 现任核心管理层任职年限
- 过去5年是否发生重大更换

**资本配置记录**（按管理层任期分别标注）：
- 留存利润去向：内生扩张 / 外延收购 / 财务投资 / 偿债 / 闲置
- 事后回报：是否有大额商誉减值 / 投资亏损 / 项目烂尾
- 若管理层已更换：前任资本错配的拖累是否已出清

**关联交易检查**：
- 频繁关联交易？定价公允？利益输送迹象？

**管理层评价**：
```
管理层未更换 → [优秀 / 合格 / 损害价值]
管理层已更换且现任 < 2年 → [观察期]
管理层已更换且现任 ≥ 2年 → 以现任记录为主
```

**输出**：
```
治理风险标志：[逐项列出，标注正常/⚠️]
管理层评价：[优秀 / 合格 / 损害价值 / 观察期]
资本配置：[简要评价]
关联交易风险：[低/中/高]
```

## 维度五：MD&A 解读

**数据源**：§10（MD&A 摘要）

⚠️ MD&A 来源验证：
- "来源: PDF年报" → 可信度基线 = 中-高
- "来源: WebSearch" → 可信度基线 = 低-中，标注完整性受限
- "来源: ⚠️ 未搜索到" → 仅输出"MD&A 不可用"

**分析内容**：

(1) **经营回顾与业绩归因**
- 管理层对收入/利润变动的自我解释
- 各业务板块表现拆解

(2) **前瞻性指引**
- 未来1-2年展望（量化 vs 方向性）
- 历史指引兑现记录

(3) **资本配置意图**
- 分红政策最新表态
- 回购计划进展
- 重大投资/收购/剥离意向
- 债务管理策略

(4) **风险因素自述**
- 管理层自行披露的主要风险
- 新增风险项（vs 上期）

**输出**：
```
MD&A 可信度：[高/中/低]
关键发现：[≤3条]
前瞻指引：[有量化/仅方向性/无]
对投资判断的影响：[正面/中性/负面]
```

## 维度六：控股结构分析（条件触发）

**触发条件**：
- 母公司持有上市子公司/联营公司重大股权（≥10%）
- 公司以"投资控股"描述主业
- 市场归类为控股公司/多元化集团
- Warnings 中有 `[结构复杂|中]`

若不适用 → 输出"不适用（非控股结构）"并跳过。

**数据源**：§9B, §4P, §1, data_pack_report SUB

**分析内容**：

(1) **SOTP 与市值对比**

```
第1步：列出上市子公司/联营公司
  数据来源优先级：§9B > data_pack_report SUB > §4P + WebSearch
  | 子公司名称 | 代码 | 持股比例 | 子公司市值 | 持股价值 | 来源 |

第2步：母公司层面净现金/净负债
  优先使用 §4P 母公司单体报表
  母公司有息负债 LL = 短期+长期有息负债
  母公司现金 MM = 现金及等价物+短期投资
  母公司净现金 = MM − LL

  若 §4P 不可用 → 使用合并口径，标注子公司负债双重扣除风险

第3步：SOTP = Σ(子公司持股价值) + 母公司净现金
第4步：控股折价率 = (SOTP − 母公司市值) / SOTP
第5步：主营隐含估值 = 母公司市值 − 可见持股价值 − 母公司净现金
```

(2) **折价分解**

| 折价因子 | 合理范围 | 评估值 | 依据 |
|---------|---------|--------|------|
| 流动性折价 | 5-10% | | 母/子日均成交额比 |
| 治理折价 | 5-15% | | 关联交易、利益一致性 |
| 复杂性折价 | 5-10% | | 持股层级、交叉持股 |
| 信息不对称折价 | 3-5% | | 独立信息披露质量 |

(3) **SOTP 敏感性**

| 情景 | SOTP | 折价率 | 主营隐含估值 |
|------|------|--------|------------|
| 乐观 (×1.2) | | | |
| 基准 | | | |
| 悲观 (×0.7) | | | |

**输出**：
```
控股结构：[适用/不适用]
SOTP 估值：[值] 百万元
控股折价率：[值]%
主营隐含估值：[值] 百万元
合理折价估计：[值]%
超额折价：[值] pct
母公司负债口径：[母公司单体/合并（降级）]
控股类型：[纯控股平台 / 混合型 / 自营为主]
```
