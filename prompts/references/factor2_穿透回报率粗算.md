基于报表利润的自上而下快速近似，用于早期筛选。若粗算回报率已远低于门槛，则直接否决，不进入因子3精算。

判断规则：
- 自由现金流过去3-5年是否持续为正？
- 融资活动现金流是否持续为正（持续借债）？
- 派息后现金储备是否仍在净增长？

```
若自由现金流长期为负 且 融资活动持续为正 → 判定为伪穿透回报 → 一票否决

注意：分配能力否决与因子1B模块六（管理层评价）是独立判断：
  - 模块六评估管理层的资本配置*意愿和质量*（是否损害价值）
  - 步骤2评估企业的现金流*生成能力*（能否产生可分配现金）
  - 管理层评价"合格"不能豁免分配能力否决
  - 反之，分配能力通过也不能豁免管理层否决
  - 两者均通过方可继续
```

## 步骤4：否决门判断

```
Rf 从 data_pack §14 读取。
门槛 II 按因子4规则计算：
  港股 = max(5%, Rf + 3%)
  A股 = max(3.5%, Rf + 2%)
  美股 = max(5%, Rf_US + 3%)

粗算快速否决规则：

① 若 R < Rf（无风险利率）→ 直接否决，不进入因子3
   理由：粗算回报率连无风险都跑不赢，精算不可能逆转

② 若 Rf ≤ R < II × 0.5（不足门槛一半）→ 直接否决，不进入因子3
   理由：因子3的现金质量修正通常在 ±2pct 以内，无法弥补如此大的差距

③ 若 II × 0.5 ≤ R < II → 标注"因子2粗算边际不达标"，继续进入因子3
   理由：因子3的现金分析可能修正（如扣除项中成长性支出占比高）

④ 若 R ≥ II → 正常通过
```
