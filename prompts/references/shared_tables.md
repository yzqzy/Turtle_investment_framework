## 支付率计算规则

```
支付率 = 年报同币种计算：各财年股息总额(报表币种) / 归母净利润(报表币种)
数据来源：data_pack_market §6 股息历史 + §3 归母净利润

⚠️ 禁止使用 yfinance 的 payoutRatio、dividendRate、trailingEps 字段
⚠️ 必须使用同币种数据（报表币种股息 / 报表币种利润）

支付率序列：过去3~5年逐年计算
支付率均值 M = 近3年均值
支付率锚定值 = 承诺支付率（若有明确承诺）；若无承诺则使用 M（近3年均值）

注销型回购 O = 过去3年年均注销型回购金额（百万元），来源 §15
稀释净效果 = 回购减少股本 − 新发行增加股本
```

## 穿透回报率公式

```
港股：
  穿透回报率 = [基准值 × 支付率锚定值 × (1 − Q%) + O] / 当前市值

A股（长期持有，Q=0）：
  穿透回报率 = [基准值 × 支付率锚定值 + O] / 当前市值

其中：
  因子2 基准值 = C（归母净利润，报表口径）→ 输出 R%
  因子3 基准值 = AA（真实可支配现金结余）→ 输出 GG%

⚠️ 税基差异说明：
  因子2 对报表利润 C 征税（粗算，快速近似）
  因子3 对真实可支配现金 AA 征税（精算，保守底线）
  两者偏差 HH = R − GG 是粗算/精算方法论差异的正常结果
  HH 通常在 1-3 pct 范围内；若 |HH| > 3 pct 需调查成因
```

## 门槛值 II 计算公式

```
Rf = data_pack §14 无风险利率

港股门槛 = max(5%, Rf + 3%)
A股门槛 = max(3.5%, Rf + 2%)
美股门槛 = max(5%, Rf_US + 3%)（Rf_US = 美国十年期国债收益率）
```

## 跨币种处理规则

穿透回报率公式中所有项必须使用同一币种：

```
基准币种 = data_pack_market 报表币种（§头部标注）

- 报表币种 = HKD → 市值用港元市值（§1 直接取值）
- 报表币种 = USD → 市值用美元市值（§1 直接取值）
- 报表币种 = CNY 但股价为 HKD：
    市值 = 港元市值 / 汇率（取 §1 或 data_pack 头部汇率）
    转换为人民币市值后计算
- 报表币种 = CNY 但股价为 USD：同理，使用汇率转换

⚠️ 禁止混用：如用人民币利润除以港元市值（可导致 5-10% 计算偏差）
```
