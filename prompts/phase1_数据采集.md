### §13：Warnings（综合生成）

在完成 §7、§8、§10 的采集后，结合已有 data_pack_market.md 中 tushare_collector.py 生成的数据（包括 §14 无风险利率、§15 回购、§16 质押），综合评估并生成 Warnings。

> **注意**：tushare_collector.py 已在 §13.1 中自动生成了部分 Warnings（数据缺失/数据异常/审计风险/高商誉/高负债类）。Agent 应**合并补充**（如治理风险等定性 Warnings），不要覆盖已有 Warnings。

**Warnings 生成规则**：

| 类型 | 触发条件 |
|------|---------|
| `[数据缺失\|高]` | §1-§6 中关键科目（净利润、OCF、Capex）标注了 `⚠️缺失` 或 `⚠️ Tushare无数据` |
| `[数据缺失\|中]` | 非关键科目缺失 |
| `[数据异常\|高]` | 某年度数据与前后年度偏差超过 300%（可能为数据源错误） |
| `[数据异常\|中]` | 某年度数据与前后年度偏差超过 100% |
| `[审计风险\|高]` | §7 搜索到审计意见异常或频繁更换审计师，或 §12 fina_audit 中审计意见非标准 |
| `[治理风险\|高]` | §7 搜索到财务造假/重大违规记录 |
| `[治理风险\|中]` | §7 控股股东大比例质押或持续减持（参考 §16 质押数据） |
| `[回购信号\|低]` | §15 回购数据显示有回购行为，需结合 §7 WebSearch 确认动机和进展 |
| `[结构复杂\|中]` | §1 确认为控股公司/多元化集团结构（触发 §9B 子公司识别） |
| `[跨币种\|中]` | 报表币种与交易币种不一致（如港元报表但人民币计价） |

输出格式：
```markdown
## 13. ⚠️ Warnings

> Phase 1 在采集过程中发现的异常信号，供 Phase 3 分析时参考。

| # | 类型 | 严重程度 | 描述 |
|---|------|---------|------|
| 1 | {类型} | {高/中} | {描述} |
```

若无任何异常，写入：「无异常发现」
